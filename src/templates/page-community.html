Meet no protocol barrier

We built VictoriaMetrics compatible with all the major protocols to give you the chance to benefit from both push and pull. No matter which protocol you use, you get the highest possible speed.

  • Graphite, OpenTSDB, and Influx protocols for push
  • EPrometheus protocol for native pull via vmagent
  • Protocol equality

Keep your data safe no matter what

If one of the servers is not responding or the whole cluster is unavailable—your data will be safe with VictoriaMetrics.

  • Replication data support
  • Incremental backups to local disk or object storage (GCS or S3-compatible)
  • Automatic stream restoration from the interruption point

  • Prometheus alerting rules definition format support

  • Keeps the alerts state on restarts

Own your data and migrate easily

No one can choose the right monitoring solution at first try. Whenever you make the right choice, VictoriaMetrics will provide various import and export options for you.

  • Data import via JSON, CSV, Prometheus exposition and native format
  • Data export via JSON, CSV, and native format
  • Backfilling via Graphite, OpenTSDB and Influx line protocols
